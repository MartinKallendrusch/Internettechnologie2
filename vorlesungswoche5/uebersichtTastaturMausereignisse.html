Ereignisse bei Interaktion mit der Maus

Ereignis Erklärung Objekttyp
click Wird ausgelöst, wenn über einem Element die Maustaste
gedrückt und wieder losgelassen wurde.		 MouseEvent
dblclick Wird ausgelöst, wenn über einem Element die Maustaste
zweimal gedrückt und wieder losgelassen wurde.		 MouseEvent
contextmenu Wird ausgelöst, wenn ein Kontextmenü (in der Regel über die
rechte Maustaste) geöffnet wurde.		 MouseEvent
mousedown Wird ausgelöst, wenn über einem Element die Maustaste gedrückt wurde. MouseEvent
mouseup Wird ausgelöst, wenn über einem Element die Maustaste losgelassen wurde. MouseEvent
mousemove Wird ausgelöst, wenn die Maus verschoben wurde.
 MouseEvent
mouseover Wird ausgelöst, wenn die Maus über ein Element bewegt wird. MouseEvent
mouseout Wird ausgelöst, wenn die Maus von einem Element fortbewegt wird. MouseEvent
mouseenter Wird ausgelöst, wenn die Maus über ein Element bewegt wird. MouseEvent
mouseleave Wird ausgelöst, wenn die Maus von einem Element fortbewegt
wird		 MouseEvent

Alle Mausereignisse stellen die nachfolgenden Eigeschaften zur Verfügung:

Eigenschaft Beschreibung
altKey Enthält die boolesche Angabe darüber, ob die (Alt)-Taste bei Auslösen des Ereignisses gedrückt war.
button Enthält die Nummer der Maustaste, die bei Auslösen des Ereignisses gedrückt war.
buttons Enthält eine Zahl, die die gedrückten Maustasten repräsentiert, die bei Aus- lösen des Ereignisses gedrückt waren. Hierbei repräsentiert die Zahl 1 die linke Maustaste, die Zahl 2 die rechte Maustaste, die Zahl 4 das Mausrad bzw. die mittlere Maustaste, die Zahl 8 die vierte Maustaste und die Zahl 16 die fünfte Maustaste. Wurden bei Auslösen des Ereignisses mehrere Tasten gedrückt, enthält buttons die Summe der jeweiligen Zahlen.
clientX Die x-Koordinate (bezogen auf den DOM-Inhalt), an der sich der Maus- zeiger bei Auslösen des Ereignisses befindet
clientY Die y-Koordinate (bezogen auf den DOM-Inhalt), an der sich der Maus- zeiger bei Auslösen des Ereignisses befindet
movementX Die x-Koordinate relativ zu der vorherigen x-Koordinate, die bei Auslösen des letzten mousemove-Ereignisses auftrat.
movmentY Die y-Koordinate relativ zu der vorherigen y-Koordinate, die bei Auslösen des letzten mousemove-Ereignisses auftrat.
region Enthält die ID der Region (bzw. des Elements), auf die sich das Ereignis bezieht.
screenX Die x-Koordinate (bezogen auf den Bildschirm), an der sich der Mauszeiger bei Auslösen des Ereignisses befindet.
screenY Die y-Koordinate (bezogen auf den Bildschirm), an der sich der Mauszeiger bei Auslösen des Ereignisses befindet.
   
   
   

Ereignisse bei Interaktion mit Tastatur und Textfeldern

Ereignis Erklärung Objekttyp
keydown Wird ausgelöst, wenn eine Taste gedrückt wurde. Winrd eine Taste längere Zeit gedrückt, wird das Ereignis mehrmals ausgelöst. KeyboardEvent
keyup Wird ausgelöst, wenn eine Taste losgelassen wurde. KeyboardEvent
keypress Wird ausgelöst, wenn per Tastatur ein Zeichen eingefügt wird. KeyboardEvent
     
     
     

Ereignisse beim Arbeiten mit Formularen

Ereignis Erklärung Objekttyp
input Wird ausgelöst, wenn der Wert einen inputs- oder eines textarea-Elements geändert wurde. Event
change Wird ausgelöst, wenn der ausgewählte Wert einer Auswahlliste , einer Checkbox oder einer Gruppe von Radiobuttons geändert wurde. Event
submit Wird ausgelöst, wenn ein Formular abgeschickt wurde. Event
reset Wird ausgelöst, wenn ein Formular zurückgesetzt wurde. Event
     
     

 

Ereignisse beim Fokussieren von Elementen

Ereignis Erklärung Objekttyp
focus Wird ausgelöst, wenn ein Element den Fokus bekommt. FocusEvent
blur Wird ausgelöst, wenn ein Element den Fokus verleirt FokusEvent
focusin Wird ausgelöst, wenn ein Element den Fokus bekommt. FokusEvent
focusout Wird ausgelöst, wenn ein Element den Fokus verleiert. FokusEvent
